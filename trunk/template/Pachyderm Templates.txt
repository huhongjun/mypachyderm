
一、Pachyderm 功能
	1.1 帐号管理
		
	1.2 媒体管理
		上载方式
			单个文件
			批量上载
				单个zip文件
				手工选择多个文件
			私有文件只有上载者可用
		文件类型：
			图片（gif、jpg）	
				zoom 模板 2560×2548
				pachyderm会收缩或扩展图片以适应分配的空间，建议采用最大的尺寸
			audio（mp3）
			Flash（swf）	320×240
			video（mov）	320×240
		媒体搜索
			独立功能
			页面设计中选择
		媒体文件元数据
			标题、描述、版权、关键字【类型、主题、日期】
			作者/发布者、贡献者、关系(与其他媒体)
			tombstone（标签） - 显示时会被编辑页面时的caption覆盖
	1.2 模板管理
		pachyderm会为显示标题、caption和其他文字设置字体
		presentation：
			可以有多个screen；每个screen可以采用不同的template；
			有些screen不能链出，如Zoom、Enlargement
			screen（home）=》screen（other）
			screen不能拷贝到别的presentation

二、模板介绍
	1、Aspects Templates（直译：覆盖同一主题的几个切面）
		1.1 介绍 
		多图界面，
			常用于展示不同概念、人或物体（用缩略图代表）之间的关系，
			也善于组合多个媒体文件来展示一个中心主题的不同方面；
			显示相关的图片和释义文字，很理想
		总计6个缩略图围绕中央的图片框和文本框，点击缩略图导致中央显示大图和相关文本；
		tombstone信息将随同缩略图或大图显示，如果caption没有设置，否则会覆盖；
		没有对外的链接（除了back按钮）
		不必6个全用
		
		1.2 编辑
		A：显示标题，55个字符，约10个单词
		B-G：
			1、图片：主图片，缩略图（可以是同一个）
			2、视频：只能是.mov或.swf
			3、主图片文本：在图片下方显示，最多45字符，约9个单词
			4、缩略图文本：40字符，9个单词
			5、切面文本：几段，可翻页
		1.3 样例：
			同一建筑，雕塑，动物等的多个视图或角度，带每个方面的简单描述
			用视频或动画展示单个主题或现象的不同方面
			6个有序的图片展示成长或变化
			6个相关的带文本注释项目展示相似与不同

	2、Commentary Template
		2.1 介绍
			允许相当详细的针对image的文本描述，提供位置展示相关材料；
			左上位置为视频；中间可翻页文本；右侧三个补充的缩略图可链接到其他screen或媒体文件或url
			也有显示标题和子标题
			
		2.2 编辑
			A、screen标题和子标题
				1.标题（最多50字符，约9个单词）；
				2.子标题（最多165字符，两个短句或一个长句）
			B、主媒体区
				1、图片
				2、电影：video或screen链接，只能是.mov或.swf
				3、图片文本：最多70字符，约13单词（覆盖 tombstone）
				4、链接：screen或url（如果B2放了视频，此处失效）
			C、主文本
				多个段落，可翻页。
			D-F、媒体条目
				1、缩略图
				2、电影：视频文件（每个条目可以有一个视频或screen链接，不能都有）
				3、字幕：显示在图片下（27字符，约6个单词）
				4、screen链接：screen或url之一
		2.3 样例
			展示单个主题的不同观点，采用视频对话或解说词
			作为目录screen（如PhoneDial）与更详细screen的中间screen，从四个区域链出；
			
	3、Enlargement Template
		3.1 介绍
			最简单的模板，静态最大图片，关联字幕，数段相关文本
			没有screen链接
			用于展示关于大哥主题的大量的文字信息；
			图片最大，除了zoom模板
			
		3.2 编辑
		
		3.3 样例
			图书评论，伴随封面图片
			信件或演讲的文本，伴随作者或讲演者的照片
			？
			食谱，伴随预先做好菜的照片
			更多的解释和说明，针对Commentary或Layers screen中的缩略图

	4、Exploration Entry template
		4.1 介绍
			标题
			一张中央大图，带字幕，可链出screen（通常是zoom或enlargement）或url
			左右两列最多6个文本，链出到screen或url（每个可代表队中央图片的简短问题，点击后到另一screen，提供更详细信息或答案）
			鼠标移过文本时，放大并有音效
			常用于phonedial 数字键区的链出screen，因可提供更多详细信息
			作为presenetation的导入界面，当展示更少links和图片时
			因为更多信息是从链出的screen获得，当单张图片和翻页文本足够时不宜用本模板，建议用Enlargement或Commentary
			
			
		4.2 编辑
		
		4.3 样例
			环绕中央图片（发明、艺术品、物体、历史档案、科学发现）最多6个问题，如怎么运作、重要日期、相关人员或其他主题
			中央某人照片，6个链接代表人生的10年周期，链出的screen详细展示那些阶段
			
	5、Layers Template
		5.1 介绍
			允许关于大哥主题的很多素材在层中组合和排序；
			便于深度探索任何主题，尤其用到多个媒体时（video、images、文本）
			
			右上角的介绍文本在所有层都是不变的
			用同样的样式显示相关素材
		5.2
		5.3 样例
			展示长期项目，如金字塔建设，切分为多个阶段 - 每层一个阶段
			比较5个科学家，包括照片、简历、一段动画或视频说明一个关键发现，没人的一段讲话或工作视频
			描述艺术运动的多个维度，如包豪斯建筑学派的建筑、绘画、教育等。
			描述。。。运动，以概括形式，如60年代，层有民权，另外有越南战争，等等

	6、media focus template
		6.1 介绍
			标题
			子标题
			翻页文本
			图片，带电影
			没有外部screen链接
			和Enlargement 模板很像，，主要不同是可以播放视频，除静态图片外
		6.2 编辑
		6.3 样例
			包括一段视频，一个取自视频的静态图片，关于视屏内容的讨论或剧本
			展示植物生长的flash视频，描述文本或问题
			显示静态图片（只有图片，没有视频），包含描述文字或讨论问题
			做presentation的about author页面，含照片与简历
